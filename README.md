
1. Install dependencies:
   ```sh
   npm install
   ```
2. Setup environment variable (optional):
   - Copy `.env.example` to `.env`
   - Open `.env` and set the environment variable:

     ```sh
     VITE_API_NINJAS_KEY=your_api_key_here
     ```

3. Start the development server:
   ```sh
   npm run dev
   ```
